Online Movie
Rental & Review System

A robust web application designed to handle movie rentals, user reviews, and secure transactions. Built from scratch focusing on strong Object-Oriented Programming (OOP) architectures and scalable folder structures.

Project Overview

This project was developed as a comprehensive platform where users can browse a wide catalog of movies, read and write reviews, and rent their favorite titles seamlessly. The core objective was to apply complex backend logic while maintaining a clean, intuitive frontend interface for the end-user.

I focused heavily on defining a proper folder structure (MVC pattern) and implementing strict Object-Oriented Programming principles. This ensured the codebase remained maintainable, modular, and easy to collaborate on.

The Challenges & Solutions

Managing the rental logic alongside user cart sessions and database updates simultaneously posed a technical challenge. I solved this by implementing secure backend endpoints and integrating a reliable payment gateway simulation, ensuring data consistency across all layers of the application.
